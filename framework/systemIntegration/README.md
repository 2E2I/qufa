# QuFa 통합 시스템

- 원천데이터로 부터 Metadata 추출, Profiling, OOO, OOO 및 데이터 적재를 수행

 * [Data Reviewer - Client](https://github.com/Data-Infuser/Designer-Client) - 피드백 지원도구(프론트엔드)
